Our Open Source Tools

NIST RMF Platform

A platform to manage the NIST AI Risk Management Framework (AI RMF). Implements the NIST AI RMF as a Django web application, enabling organizations to manage, assess, and document AI risks in alignment with the NIST AI RMF core functions: Govern, Map, Measure, and Manage. Features modular Django apps, REST API, admin interface, and a responsive dashboard.

AI RMF (ISO 42001)

AI Risk Management Framework based on ISO 42001:2023 and ISO/IEC 23894. A Django-based web application for managing AI systems, risk assessments, and compliance with international standards. Features system, capability, use case, user, stakeholder, model card, and risk assessment management, with a modern responsive UI.

Doc-Sherlock

Doc-Sherlock is a security tool designed to identify hidden text in PDF documents that could be used for prompt injection attacks against LLM systems. It helps security teams detect common techniques used to hide malicious content from human reviewers while remaining accessible to LLM parsers. Includes detection for invisible text, obscured content, metadata anomalies, and more.
